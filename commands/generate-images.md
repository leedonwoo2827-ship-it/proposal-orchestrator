# /proposal-orchestrator:generate-images

제안서용 이미지를 생성하는 커맨드입니다.
기존 제안서 내용을 기반으로 적절한 이미지 프롬프트를 설계하고 이미지를 생성합니다.

## 실행 절차

### 1단계: 요청 확인
사용자에게 다음 항목을 확인하세요:
- **project_id 또는 텍스트**: 이미지의 기반이 될 제안서 내용
- **이미지 개수**: 몇 장을 생성할지
- **용도**: 표지(cover), 슬라이드(slide), 요약(summary), 결론(conclusion)
- **스타일/분위기**: 미니멀, 인포그래픽, 실사, 일러스트, 금융, 공공 등

### 2단계: 텍스트 확보
- 기존 project_id가 있으면 해당 프로젝트의 관련 텍스트를 가져옵니다
- 텍스트가 직접 제공되면 그것을 활용합니다
- 요약이 필요하면 핵심 키워드와 메시지를 추출합니다

### 3단계: 프롬프트 설계
- image-prompt-design 스킬의 가이드라인에 따라 프롬프트를 설계합니다
- 용도별 템플릿을 적용합니다
- 분위기별 키워드를 추가합니다
- 설계된 프롬프트를 사용자에게 보여주고 확인 받습니다

### 4단계: 이미지 생성
- text2img-mcp를 호출하여 이미지를 생성합니다
- 여러 장인 경우 순차적으로 생성하며 진행 상황을 보고합니다

### 5단계: 결과 보고
- 생성된 이미지 목록과 설명을 제공합니다
- 각 이미지에 대한 캡션을 제안합니다
- 추가 이미지 생성 또는 재생성 여부를 물어봅니다

## 사용 예시
```
/proposal-orchestrator:generate-images
이 제안서 요약을 기반으로 썸네일 이미지 3개 만들어 줘.
분위기는 금융/투자 느낌으로.
```
