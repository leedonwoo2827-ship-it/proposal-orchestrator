# /proposal-orchestrator:watch-status

현재 프로젝트의 진행 상황을 점검하고 보고하는 커맨드입니다.
텍스트 작성 현황, 이미지 생성 현황, 누락 사항을 종합적으로 분석합니다.

## 실행 절차

### 1단계: 프로젝트 확인
- project_id를 확인합니다
- 명시되지 않은 경우 가장 최근 작업한 프로젝트를 기준으로 합니다

### 2단계: 텍스트 상태 조회
현재까지 작성된 내용을 분석합니다:
- 각 섹션의 작성 상태 (완료 / 부분 작성 / 미작성)
- 현재 총 글자 수 vs 목표 글자 수
- 섹션별 분량 균형 확인

### 3단계: 이미지 상태 조회
생성된 이미지를 분석합니다:
- 이미지 목록 (용도, 경로, 생성 일시)
- 어떤 섹션에 이미지가 배정되었는지
- 미생성 이미지 확인

### 4단계: 종합 보고
다음 형식으로 보고합니다:

```
[프로젝트 종합 현황]
- Project ID: PROP-2026-XXX
- 텍스트 진행률: XX%
- 이미지 진행률: X/Y장
- 작성 완료 섹션: ...
- 미작성 섹션: ...
- 다음 권장 작업: ...
```

### 5단계: 보완 제안
분석 결과에 기반하여 구체적인 보완 작업을 제안합니다:
- 분량이 부족한 섹션
- 이미지가 없는 중요 섹션
- 구조적으로 빠진 내용
- 다음 우선순위 작업

## 사용 예시
```
/proposal-orchestrator:watch-status
지금까지 만든 제안서 구조와 이미지 목록을 정리해서 보여줘.
빠진 섹션이 있는지 같이 검토하자.
```
