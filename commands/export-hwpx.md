# /proposal-orchestrator:export-hwpx

작성된 제안서를 HWPX 파일로 내보내는 커맨드입니다.
hwpx-writer MCP를 활용하여 본문 텍스트와 이미지를 결합한 최종 파일을 생성합니다.

## 실행 절차

### 1단계: 프로젝트 확인
- project_id를 확인합니다
- 현재 작성 상태를 간단히 점검합니다
- 미완성 섹션이 있으면 사용자에게 알립니다

### 2단계: 출력 옵션 확인
사용자에게 다음을 확인합니다:
- **출력 파일명**: 기본값은 project_id 기반
- **이미지 포함 여부**: 생성된 이미지를 문서에 삽입할지
- **페이지 레이아웃**: A4, 양면/단면, 여백 설정
- **스타일 템플릿**: 제안서 양식에 맞는 서식 적용 여부

### 3단계: 서식 변환 (색상 및 볼드 처리)
본문 텍스트를 HWPX에 전달하기 전에 다음 서식을 적용합니다:

#### 출처별 색상 변환
- `<source type="rawdata">텍스트</source>` → **적색 글자 (Red, #FF0000)** 로 변환
- `<source type="reference">텍스트</source>` → **녹색 글자 (Green, #008000)** 로 변환
- 태그가 없는 텍스트 → 기본 검정색 유지

#### 마크다운 서식 변환
- `**텍스트**` → Bold (굵게) 서식 적용, `**` 기호는 제거
- `*텍스트*` → Italic (기울임) 서식 적용, `*` 기호는 제거
- `***텍스트***` → Bold + Italic 서식 동시 적용

#### 복합 서식
- 색상과 볼드가 동시에 적용될 수 있음 (예: rawdata 출처의 제목 → 적색 + 굵게)
- source 태그 내부의 마크다운 볼드도 정상적으로 변환

### 4단계: HWPX 생성
- hwpx-writer MCP를 호출합니다
- 본문 텍스트, 이미지 정보, 스타일 옵션, **색상/서식 정보**를 전달합니다
- 생성 진행 상황을 보고합니다

### 5단계: 결과 안내
- 생성된 파일 경로를 안내합니다
- 파일 크기, 페이지 수 등 기본 정보를 제공합니다
- 수정이 필요한 경우 다음 단계를 안내합니다

## 사용 예시
```
/proposal-orchestrator:export-hwpx
지금까지 작성한 내용으로 HWPX 파일 만들어줘.
이미지도 같이 넣어줘.
```
